# example-react


## 启动

```bash
npm start
```

## 打包

```bash
npm run build
```

## test

```bash
npm run test
```

## 代码风格检查

```bash
npm run lint
```

## 配置

该项目是基于 caiyi-cli<br >
caiyi-cli 会根据项目根目录 `.caiyi`文件配进行构建，如需修改配置请查看[文档](http://vvo.io/cli/configuration)
